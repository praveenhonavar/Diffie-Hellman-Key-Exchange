# CG-DIFFE


## Steps:


1) Enter private key for Alice:
   5 (any prime number)

2)  Enter private key for Bob:
   3 (any prime number)


3) Right click anywhere on the screen to get menu

4)  * Exchange keys
    * Shared Keys


If you get *connection established* they excecuted properly
